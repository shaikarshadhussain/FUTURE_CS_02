# Phishing Email Detection & Awareness System

## Overview
This repository contains a phishing email analysis and awareness report created as part of Cyber Security Task 2 (2026) by Future Interns.

## Tools Used
- Google Message Header Analyzer
- MXToolbox
- Browser-based link inspection

## Analysis Approach
- Studied phishing email samples
- Analyzed headers and sender domains
- Identified phishing indicators
- Classified email risk
- Created awareness guidelines

## Purpose
Educational and awareness-focused.
No hacking or illegal activity involved.
